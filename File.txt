# AI-Powered Resume Analyzer Project Prompt

Act as a Senior Full-Stack Developer, AI Engineer, and UI/UX Designer.

Build a complete production-ready web application called **"AI-Powered Resume Analyzer"**.

## Project Objective

The system should help users analyze their resumes using AI, check ATS compatibility, compare resumes with job descriptions, identify missing skills, generate interview questions, and provide improvement suggestions.

The project should be suitable for a final-year college minor project and demonstrate AI, NLP, full-stack development, and modern UI design.
